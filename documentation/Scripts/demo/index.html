Demo Files

All demo files used in the theme is saved at assets/demo/. The filenames correspond to the html file in which the demo is used.

A notable exception is the demo.js file which houses the demo scripts that are being called by the demo constantly.

Styling of the plugins were mostly done through modifying the available code with the apps.less file
