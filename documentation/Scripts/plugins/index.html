  • Plugins

Paper comes packed with many different third-party plugins that has been customized fit and work with the template. While we do not provide support on the third party plugins, one can go to the site of the plugins and request support individually.

Plugin Name (in folder) Description
bootbox Bootbox.js is a small JavaScript library which allows you to create programmatic dialog boxes using Bootstrap modals, without having to worry about creating, managing or removing any of the required DOM elements or JS event handlers.
bootstrap-datepicker A datepicker for Bootstrap
bootstrap-datetimepicker Date and time picker for Bootstrap
bootstrap-switch Turn checkboxes and radio buttons in toggle switches.
bootstrap-tabdrop Responsive Tabs
bootstrap-timepicker A simple timepicker component for Twitter Bootstrap
bootstrap-touchspin A mobile and touch friendly input spinner component for Bootstrap 3. It supports the mousewheel and the up/down keys.
bootstro.js Show your users a guided tour of what is what on a page.
card Make your credit card form better in one line of code.
chartjs Simple, clean and engaging charts for designers and developers
nvd3 Build re-usable charts and chart components for d3.js
flotcharts Flot is a pure JavaScript plotting library for jQuery, with a focus on simple usage, attractive looks and interactive features.
morrisjs Good looking simple charts
clockface Timepicker for Bootstrap
codeprettifier A Javascript module and CSS file that allows syntax highlighting of source code snippets in an html page.
datatables Supercharges your tables
bootstrap-datepaginator A jQuery plugin which takes Twitter Bootstrap's already great pagination component and injects a bit of date based magic. In the process creating a hugely simplified and modularised way of paging date based results in your application.
dropzone DropzoneJS is an open source library that provides drag'n'drop file uploads with image previews.
bootstrap-duallistbox Bootstrap Dual Listbox is a responsive dual listbox widget optimized for Twitter Bootstrap.
easy-pie-chart Easy pie chart is a jQuery plugin that uses the canvas element to render simple pie charts for single values.
autosize Small jQuery plugin to allow dynamic resizing of textarea height, so that it grows as based on visitor input.
bootstrap-colorpicker Simple Colorpicker
bootstrap-daterangepicker Date range picker component for Bootstrap
bootstrap-datetimepicker Date and time picker for Bootstrap
fseditor jQuery Fullscreen Editor
jquery.inputmask An inputmask helps the user with the input by ensuring a predefined format. This can be useful for dates, numerics, phone numbers, etc.
jasny-fileupload The file input plugin allows you to create a visually appealing file or image input widgets
bootstrap-markdown Markdown editing meet Bootstrap.
multi-select Drop-in replacement for the standard select with multiple attribute activated.
Nestable Drag & drop hierarchical list with mouse and touch compatibility (jQuery plugin)
pagedown-bootstrap A jQuery plugin that dynamically generates a WYSIWYG editor for Markdown with integrated Twitter Bootstrap styling.
parsleyjs Javascript form validation library
select2 Select2 is a jQuery based replacement for select boxes. It supports searching, remote data sets, and infinite scrolling of results.
stepy A jQuery wizard plugin
boostrap-tokenfield Advanced tagging/tokenizing plugin for jQuery and Twitter Bootstrap with a focus on keyboard and copy-paste support.
typeahead.js A flexible JavaScript library that provides a strong foundation for building robust typeaheads
jquery-validation This jQuery plugin makes simple clientside form validation easy, whilst still offering plenty of customization options. This plugin is needed if you are using the the stepy for form wizard validation. Otherwise we recommend Parsley
wysihtml5 wysihtml5 is an open source rich text editor based on HTML5 technology and the progressive-enhancement approach.
x-editable This library allows you to create editable elements on your page.
fullcalendar Full-sized Calendar Plugin
gridforms Gridforms is a front-end library which handles the boilerplate necessary to build grid based forms.
gmaps Google Maps API made simple
iCheck Highly customizable checkboxes and radio buttons for jQuery and Zepto
ion.rangeSlider Cool, comfortable and easily customizable range slider
Jcrop Jcrop is the quick and easy way to add image cropping functionality to your web application
editable This plugin makes it possible to edit the content of an element simply by double clicking on it.
mousewheel A jQuery plugin that adds cross-browser mouse wheel support with delta normalization.
jqvmap JQVMap is a jQuery plugin that renders Vector Maps.
knob Nice, downward compatible, touchable, jQuery dial.
pnotify JavaScript notifications for Bootstrap
skylo A Twitter Bootstrap extension to add progress loader at the top of the page.
jquerypulsate jQuery.pulsate.js adds a pulsating effect to elements. Useful for drawing the users attention.
quicksearch A jquery based plug-in for filtering large data sets with user input
Shufflejs Categorize, sort, and filter a responsive grid of items
simpleWeather A simple jQuery plugin to display current weather data for any location and doesn't get in your way.
jquery.sparkline jQuery plugin that generates small inline charts directly in browser
summernote Super Simple WYSIWYG Editor on Bootstrap
datatable-fixedheader The FixedHeader plug-in for DataTables will float the 'thead' element above the table at all times
velocity.js An incredibly fast animation engine for motion designers.
