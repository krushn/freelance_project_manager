LESS Files

The theme is built using LESS. However you should precompile the files in CSS for improved performance in production. We recommend WinLESS in the absence of a Node.js compiler.

All the .less files are located in assets/less/ directory of the admin theme and are called via styles.less file. Paper code and Bootstrap code are kept separate so that you can easily switch to a new Bootstrap version by overwritign the files at assets/less/bootstrap/.

File
apps.less Customization of all the different plugins used by Paper
colorschemes.less The styles for different color schemes available with Paper as proerty of a class
demo.less Some styles used in displaying the demo but not needed for a fresh install
grid-sh.less Bootstrap 3 responsive columns of same height
helpers.less Helper classes for Paper
layout.less Handles the different layout options of the theme.
leftbar.less The left sidebar navigation and search
material-colors.less Color variables used in the theme from Google's Material Design
navbar.less Custom styles overwriting Bootstrap default navbar
panels.less Completely redone panels to allow more options
panels-custom.less The styles for the custom panels - Recent Activities, Contact List, Tasks, etc.
tabs.less The CSS for the inline tabs and accordions
tiles.less the Tiles component
timeline.less The CSS for Timeline page
topnav.less The topnav dropdowns
variables.less Bootstrap's variables.less with additional variables for Paper
extend-mixins.less Mixins used in Paper and in some cases modification of bootstrap's mixins.
